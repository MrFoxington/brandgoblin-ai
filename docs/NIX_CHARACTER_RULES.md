## Generation Rules

**Before generating ANY image:**
1. Load `/public/nix/nix-master-sheet.png`
2. Use it as the PRIMARY reference image at maximum strength
3. Describe ONLY the pose/action — never describe the character design
4. After generation, run the quality checklist below

**Quality Checklist (all must be YES):**
- [ ] Same face shape?
- [ ] Same eyes?
- [ ] Same smile?
- [ ] Same hoodie?
- [ ] Same colors?
- [ ] Same proportions?
- [ ] Same personality?
- [ ] Feels like same character?

**If any answer is NO → REJECT and REGENERATE.**
