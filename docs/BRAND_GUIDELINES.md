## Colors

| Name | Hex | Usage |
|---|---|---|
| Royal Purple | `#7C3AED` | Primary — buttons, accents, CTAs |
| Emerald Green | `#10B981` | Secondary — highlights, success states |
| Warm Gold | `#FBBF24` | Accent — stars, badges, wand |
| Dark BG | `#0F172A` | App background |
| Card BG | `#1E1B2E` | Card surfaces |
| Text Primary | `#FFFFFF` | Headlines |
| Text Muted | `#94A3B8` | Body, secondary text |
| Text Faint | `#475569` | Placeholders, disabled |
| Light | `#F8FAFC` | Light mode bg |
| Soft Purple | `#E9D5FF` | Light mode accents |

---

## Typography

- **Display / Headlines:** Font Display (extrabold)
- **Body:** System sans-serif
- **Gradient Text:** Purple → Green (`#7C3AED` → `#10B981`)

---

## Logo Usage

- Use `/public/logos/brandgoblin-logo.png` on dark backgrounds
- Use `/public/logos/brandgoblin-logo-light.png` on light backgrounds
- Never stretch, recolor, or modify the logo
- Minimum size: 40px height

## UI Principles

1. Dark backgrounds — deep navy/dark purple
2. Purple + green glow effects
3. Card-based layout with subtle borders
4. Nix appears on key emotional touchpoints
5. Sparkles and magical effects reinforce the brand
6. Never cold, sterile, or enterprise-feeling
