# 🪄 BrandGoblin AI

> Type an idea. Summon a brand.

BrandGoblin AI turns a single business idea into a complete, launch-ready brand kit:
10 name options, a recommendation, taglines, brand story, brand voice, a mascot concept,
a logo prompt, a 5-color palette, website copy, a full social media kit, viral
marketing/meme ideas, and a 7-day launch plan — all generated by Claude in under a minute.

## Tech stack

- **Next.js 14** (App Router) + **TypeScript**
- **Tailwind CSS** — dark, glowing, purple/emerald "magic startup" theme
- **Supabase** — auth + Postgres database (with RLS)
- **Anthropic Claude API** — brand kit generation
- **Stripe** — subscription checkout + webhook scaffolding (placeholder, ready to wire up)
- Deploys cleanly to **Vercel**

## 1. Install dependencies

```bash
npm install
```

## 2. Set up Supabase

1. Create a project at [supabase.com](https://supabase.com).
2. In the SQL editor, run [`supabase/schema.sql`](supabase/schema.sql). This creates:
   - `public.users` (mirrors `auth.users`, adds `credits` + `plan`, auto-populated via trigger on signup)
   - `public.brand_generations` (stores every generated brand kit)
   - Row Level Security policies so users can only see/edit their own data
3. In **Project Settings → API**, grab your `Project URL`, `anon public` key, and
   `service_role` key.
4. (Optional) In **Authentication → Providers**, you can disable "Confirm email" for
   faster local testing.

## 3. Configure environment variables

```bash
cp .env.example .env.local
```

Fill in:

- `NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY`, `SUPABASE_SERVICE_ROLE_KEY`
- `ANTHROPIC_API_KEY` (get one at [console.anthropic.com](https://console.anthropic.com))
- Stripe keys, if/when you wire up billing (`STRIPE_SECRET_KEY`, `STRIPE_WEBHOOK_SECRET`,
  `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY`, `STRIPE_PRICE_ID_PRO`, `STRIPE_PRICE_ID_AGENCY`)

## 4. Run locally

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000).

## 5. Try it

1. Sign up (you start with 3 free credits).
2. Go to **/generate**, describe a business idea, pick a vibe, and submit.
3. Claude generates a full 12-section brand kit, saved automatically to your vault.
4. View it anytime at **/dashboard** → click into any card.

## How AI generation works

- [`src/lib/prompts.ts`](src/lib/prompts.ts) defines `BRAND_GOBLIN_SYSTEM_PROMPT`
  (the agency/strategist/copywriter persona) and `buildBrandKitPrompt()`, which injects
  the user's form inputs and a strict JSON schema.
- [`src/app/api/generate/route.ts`](src/app/api/generate/route.ts) checks the user's
  credits, calls the Anthropic API (`claude-sonnet-4-6`), parses the JSON response into
  a `BrandKit` (see [`src/types/index.ts`](src/types/index.ts)), saves it to
  `brand_generations`, and decrements credits for free-plan users.
- Swap in OpenAI by replacing the Anthropic client call in that route — the prompt
  builder and JSON schema are provider-agnostic.

## Credits & plans

- **Free** — 3 brand generations, full kit each time.
- **Pro** ($19/mo placeholder) — unlimited generations, unlimited saved brands.
- **Agency** ($49/mo placeholder) — everything in Pro + client-ready, bulk, white-label
  output (extend the prompt/UI to add a "client mode" toggle).

Credits and plan live on `public.users`. The Stripe webhook
([`src/app/api/stripe/webhook/route.ts`](src/app/api/stripe/webhook/route.ts)) upgrades
a user's `plan` and credits on `checkout.session.completed` — wire up real Stripe price
IDs and point your Stripe webhook at `/api/stripe/webhook` to go live.

## Deploying to Vercel

1. Push this repo to GitHub.
2. Import it in [Vercel](https://vercel.com/new).
3. Add all the environment variables from `.env.example` in the Vercel project settings.
4. Deploy. Update `NEXT_PUBLIC_APP_URL` to your production URL once you have one (used
   for Stripe redirect URLs and email confirmation links).
